## 🔍 Quick Reference

### Commands

```bash
# Install dependencies
npm install

# Development
npm run dev

# Build
npm run build

# Deploy
vercel
```

### URLs (After Deployment)

- Landing: `https://your-app.vercel.app/`
- Privacy: `https://your-app.vercel.app/privacy`
- Terms: `https://your-app.vercel.app/tos`
- Delete: `https://your-app.vercel.app/delete-account`

### Key Files

- Landing page: `app/page.tsx`
- Privacy Policy: `app/privacy/page.tsx`
- Terms of Service: `app/tos/page.tsx`
- Delete Account: `app/delete-account/page.tsx`
- Navigation: `components/Navigation.tsx`
- Footer: `components/Footer.tsx`
